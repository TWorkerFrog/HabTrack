# HabitTracker

[![Android](https://img.shields.io/badge/Android-16%2B-green.svg)](https://developer.android.com)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

**HabitTracker** — это мобильное приложение для отслеживания привычек с календарём и статистикой. Помогает формировать полезные привычки через системный учёт и визуализацию прогресса.

## Скриншоты

| Главный экран | Календарь | Статистика | Профиль |
|---------------|-----------|------------|---------|
| Список привычек с прогрессом | Тепловая карта по дням | Графики и диаграммы | Управление категориями |

## Возможности

### Основные
- ➕ Создание/редактирование/удаление привычек
- ✅ Ежедневные отметки выполнения
- 🎯 Прогресс-бар текущего дня
- 📦 Drag-and-drop для сортировки привычек

### Календарь
- 📅 Месячная сетка с цветовой индикацией
- 🔄 Навигация по месяцам
- 📋 Детали дня с BottomSheet

### Статистика
- 📊 Линейный график прогресса
- 🥧 Круговая диаграмма по категориям
- ⏱️ Выбор периода: день, неделя, месяц, произвольный
- 🔥 Текущая серия выполнений (streak)

### Профиль
- 🔐 Локальная авторизация
- 🔑 Смена пароля
- 🗑️ Удаление аккаунта
- 📁 Управление категориями (CRUD)

## Технологии

- **Язык:** Java
- **UI:** XML + Material Design
- **БД:** SQLite
- **Графики:** MPAndroidChart
- **Минимальная версия:** Android 16 (4.1 Jelly Bean)
